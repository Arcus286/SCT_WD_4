# Simple Todo App

This is a clean, interactive to-do list application built entirely with vanilla HTML, CSS, and JavaScript. It demonstrates fundamental DOM manipulation and event handling principles.



## üìù Features

* **Add Tasks:** Quickly add new items to your "Todo" list.
* **Edit Tasks:** Click "Edit" to modify the text of an existing task.
* **Delete Tasks:** Remove tasks you no longer need with the "Delete" button.
* **Complete Tasks:** Check the box next to a task to move it to the "Completed" list.
* **Mark Incomplete:** Uncheck a completed task to move it back to the "Todo" list.

## üíª Technologies Used

* **HTML5:** For the basic structure and content of the application.
* **CSS3:** For all styling, layout, and visual presentation.
* **Vanilla JavaScript:** For all interactivity, event handling, and DOM manipulation (no external libraries).
